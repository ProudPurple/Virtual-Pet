This is my submission for the Intro to Programming catagory for FBLA

NAVIGATION GUIDE:
To start the program simply run the .exe file in the main folder
The other files in the main folder are used as dependencies for my program to run
.vscode is my json files used for compiling and debuging my program
bin includes my resource files used to give the exe an icon and the rest of the SFML files necessary for my program
Sprites are simply all the images I drew and used in the program
Src is all of my code used for the program

CREDITS:
This entire program is made in C++ alongisde SFML, an open source API for C++ allowing for use of C++ code in displaying images on the screen
This was made and editted using VSCode
No Generative AI was used in this project for creation of or inspiration for the code or methods used
